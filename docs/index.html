ZaksClips

A personal desktop tool that automates the post-stream clip workflow for a chess content creator. It runs entirely on the creator's local machine — no server, no cloud, no multi-user access.

How it works

  1. Fetches recent clips from the Twitch API
  2. Downloads the corresponding VOD segment from YouTube
  3. Transcribes audio locally using Whisper (Apple Silicon)
  4. Burns styled captions and a title overlay onto the video with ffmpeg
  5. Uploads the finished video to YouTube Shorts and/or TikTok — only when the creator explicitly confirms

Why TikTok access is needed

ZaksClips uses the TikTok Content Posting API to upload finished videos directly to the creator's own TikTok account. The app never accesses any other user's account or data. Every upload requires explicit confirmation before it is sent.

Permissions used

  • video.upload — upload video files to the creator's account
  • video.publish — publish uploaded videos to the creator's profile

No other TikTok data is read or modified. The app does not access followers, comments, analytics, or any other user's content.

Developer

ZaksClips is built and used by a single developer for personal content creation. Source code is available at github.com/zparks/ZaksClips.
