Privacy Policy

Last updated: April 7, 2026

ZaksClips is a personal tool that runs entirely on your local machine. Here's how it handles data:

  • No data collection. ZaksClips does not collect, store, or transmit any personal data to any server. All processing happens locally.
  • API credentials (Twitch, YouTube, TikTok) are stored only on your machine in a local .env file and token files. They are never shared.
  • OAuth tokens are saved locally for convenience and are only used to authenticate with the platforms you've authorized.
  • Video content is processed locally and only uploaded to platforms when you explicitly confirm.
  • No analytics, tracking, or third-party services are used by this tool.

Questions? This is a personal project by the developer — reach out via GitHub.
